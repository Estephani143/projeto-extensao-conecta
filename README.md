# 🌐 Conecta+

### Inclusão Digital para Idosos

## 📌 Sobre o projeto

O **Conecta+** é um projeto de extensão desenvolvido com o objetivo de promover a inclusão digital de pessoas idosas, oferecendo orientações simples e acessíveis sobre o uso de tecnologias no dia a dia.

A proposta busca reduzir a dificuldade no uso de smartphones e serviços digitais, contribuindo para maior autonomia, segurança e participação social.

---

## 🎯 Objetivos

* Facilitar o uso de tecnologias básicas (WhatsApp, Pix, serviços online)
* Promover a inclusão digital de idosos
* Estimular o uso consciente e seguro da internet
* Desenvolver uma solução tecnológica com impacto social

---

## 👵 Público-alvo

* Pessoas idosas
* Iniciantes no uso de smartphones
* Usuários com dificuldade em tecnologia

---

## 💡 Funcionalidades

✔ Tutorial de uso do WhatsApp
✔ Passo a passo para realizar Pix
✔ Orientações para evitar golpes
✔ Guia para acesso a serviços do governo
✔ Botão de leitura em voz (acessibilidade)
✔ Modo de aumento de fonte (modo idoso)
✔ Interface simples e intuitiva
✔ Design acessível e responsivo

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Web Speech API (leitura em voz)
* PWA (Progressive Web App)

---

## 📱 Instalação como App

O Conecta+ pode ser instalado no celular como um aplicativo:

1. Acesse o site pelo navegador (Chrome)
2. Clique nos três pontos (menu)
3. Selecione **"Adicionar à tela inicial"**
